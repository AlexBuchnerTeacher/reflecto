# Reflecto File Size Limits

UI Screens: 150–250 Zeilen (max 300)  
Sections: 120–200 Zeilen (max 250)  
Widgets: 60–120 Zeilen (max 150)  
Logic: 150–250 Zeilen (max 300)  
Controllers: 80–140 Zeilen (max 250)  
Provider: 40–80 Zeilen (max 150)  
Models: 30–60 Zeilen (max 120)

Wenn eine Datei diese Grenze überschreitet:
→ automatisch splitten  
→ neue Datei(en) erstellen  
→ Dateinamen anhand naming_conventions.md bestimmen
