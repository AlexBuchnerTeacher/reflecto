# Reflecto Folder Structure

lib/
 └─ features/
      └─ <feature>/
          ├─ ui/
          ├─ sections/
          ├─ controllers/
          ├─ logic/
          ├─ widgets/
          └─ models/

## Regeln
1. Neue Features immer unter /features/<name>
2. Keine großen globalen Utils
3. Services zentral unter /services
